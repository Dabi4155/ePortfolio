Push Down Automata Simulator

This Python code simulates a Pushdown Automaton (PDA) for two languages: "a^n b^n" and "a^n b^m c^n". It allows the user to choose one of these languages and then provides an interface for inputting a string to be processed. The PDA operates by following a set of transition rules specific to each language, manipulating its stack as it reads the input string. The simulation checks if the string is accepted based on whether the PDA reaches the designated final state and if the stack is empty at the end. | Additionally, the code generates a visual diagram of the PDA’s transitions using the NetworkX and Matplotlib libraries.

Improved Magnetic Bubble Sort Algorithm

In our paper on the "Improved Magnetic Bubble Sort Algorithm" (IMBSA), we discuss enhancements made to the Magnetic Bubble Sort Algorithm (MBSA) by introducing optimizations such as the "last swapped" concept and early termination. These improvements are designed to reduce redundant comparisons and swaps, especially in datasets with varying levels of redundancy, thereby increasing sorting efficiency—particularly for large and redundant datasets. Through my experiments, I found that IMBSA outperforms the original MBSA in all tested redundancy levels, demonstrating significant improvements in runtime. The optimizations, such as tracking the last swap and utilizing early termination, drastically reduce unnecessary operations, making IMBSA much more efficient than MBSA.

Tower Defense Game

The Tower Defense game is a strategy-based video game where players place defensive structures, called towers, along a path to prevent waves of enemies from reaching a target, often a base. Enemies move along pre-determined paths, and the player must strategically position towers that automatically attack these enemies as they pass by. The goal is to prevent the enemies from reaching their destination while managing resources to build and upgrade towers. The gameplay usually includes various types of enemies with different strengths and weaknesses, as well as towers with unique abilities to counter those threats. Successful defense requires both planning and quick adaptation to changing enemy patterns and tower upgrades.
